# 🎬 Nic Cage Marathon Planning Checklist

Use this checklist to organize your epic Nicolas Cage marathon event!

## Phase 1: Setup & Distribution (1-2 weeks before)

- [ ] Deploy the voting site (see `FIREBASE-QUICKSTART.md`)
- [ ] Test the voting form end-to-end (submit a test vote, verify it appears in the results dashboard)
- [ ] Create a group chat or email for your friends
- [ ] Write a fun event announcement:
  - [ ] Include date and approximate time (e.g. 10 AM - midnight)
  - [ ] Mention theme (Nic Cage marathon!)
  - [ ] Share the voting link
- [ ] Decide on a voting deadline (e.g. "votes due Thursday")
- [ ] Mention if food will be potluck or provided by you

## Phase 2: Voting Period (1 week before)

- [ ] Send the voting link to all friends
- [ ] Send a reminder 2-3 days before the deadline
- [ ] Send a final reminder 24 hours before the deadline
- [ ] Monitor the results dashboard to track who has voted

## Phase 3: Results & Schedule (3-5 days before)

- [ ] Open the results dashboard
- [ ] Review the generated rankings and schedule
- [ ] Manual adjustments:
  - [ ] Reorder based on movie runtimes
  - [ ] Group similar films together
  - [ ] Schedule the TOP PICKS for evening (when most people will be there)
  - [ ] Put lighter/weirder films in early morning and late night
- [ ] Export the final schedule (CSV or copy to clipboard)
- [ ] Create a nice version to share (spreadsheet, printable, etc.)

## Phase 4: Food Planning (4-5 days before)

### Decide on Food Format

- [ ] **Option A - Potluck**: Assign friends specific items to bring
  - [ ] Appetizers/snacks for person 1
  - [ ] Main snacks for person 2
  - [ ] Drinks for person 3
  - [ ] Desserts for person 4
  - [ ] Pizza/delivery around lunch
- [ ] **Option B - You provide everything**: Plan shopping
- [ ] **Option C - Hybrid**: You provide meals, friends bring snacks

### Food Shopping/Prep

- [ ] Check allergies/dietary restrictions
- [ ] Make/buy a sweet snack
- [ ] Make/buy a savory snack
- [ ] Get drinks (water, soda, coffee, tea, juice, alcohol)
- [ ] Buy cups, plates, napkins, utensils
- [ ] Plan lunch option(s)
- [ ] Plan dinner option(s)
- [ ] Get ice if needed
- [ ] Prep snacks the day before (cut veggies, make dips, etc.)

### Themed Food Ideas

- [ ] Create a Nic Cage-themed snack (gold-dipped treats, "treasure" popcorn, etc.)
- [ ] Make a themed drink (signature cocktail or mocktail)
- [ ] Prepare snacks that fit specific movies (see README.md)

## Phase 5: Setup Day (Day of event)

### Morning (8-9 AM)

- [ ] Clean common area where you'll be watching
- [ ] Set up TV/projector and test audio
- [ ] Arrange seating (enough for expected crowd)
- [ ] Create a snack table/bar:
  - [ ] Lay out all snacks in bowls/plates
  - [ ] Set up drinks with ice
  - [ ] Provide napkins, utensils, small plates
  - [ ] Label themed snacks/drinks
- [ ] Set up bathroom access
- [ ] Create a bathroom break schedule (every 90 min)
- [ ] Print out or display the marathon schedule
- [ ] Start background music 30 min before guests arrive

### During Event

- [ ] Start first movie ON TIME (10 AM)
- [ ] Pause for bathroom/snack breaks between movies (15 min)
- [ ] Long break at lunch (30 min, around 1-2 PM)
- [ ] Dinner break (30 min, around 6-7 PM)
- [ ] Refresh snacks/drinks as needed
- [ ] Check in with guests about energy/enjoyment
- [ ] Adjust later movies if group wants changes
- [ ] Take photos for memories!
- [ ] Have a "highlight reel" ready for end (clips of best moments)

## Phase 6: Send-Off

- [ ] Thank everyone for coming
- [ ] Have a funny ending (funny quote from a Nic Cage movie, etc.)
- [ ] Ask people to rate the event (email survey after)
- [ ] Share photos/videos with the group
- [ ] Plan next theme-d marathon? 😄

## Optional Extras

- [ ] Create event hashtag for social media posts
- [ ] Make printed programs/schedules for guests
- [ ] Create a Nic Cage trivia game during breaks
- [ ] Make name tags/badges with Nic Cage references
- [ ] Play Nic Cage memes during breaks
- [ ] Create a voting board where guests rate each movie as they watch
- [ ] Have a "Nic Cage Bingo" card game
- [ ] Make a playlist of Nic Cage movie quotes

## Technical Reminders

- [ ] Test internet/streaming services before event
- [ ] Have backup streaming services (Netflix, physical DVDs, etc.)
- [ ] Keep phone chargers available for guests
- [ ] Have Wi-Fi password ready to share
- [ ] Test projector/TV with sound before guests arrive
- [ ] Have HDMI cables ready as backup
- [ ] Close unnecessary browser tabs/notifications before playing movies

## Post-Event (After Marathon)

- [ ] Clean up snacks/dishes
- [ ] Return borrowed items
- [ ] Send thank you messages to friends
- [ ] Collect feedback:
  - [ ] Which movies were hits?
  - [ ] Which were misses?
  - [ ] Would they do it again?
  - [ ] Any movie suggestions for next time?
- [ ] Update your movies.json with any new suggestions
- [ ] Archive all voting results for future events
- [ ] Celebrate your successful event! 🎉

---

## Tips for Success

✅ **Start planning early** - 2 weeks is ideal
✅ **Communicate clearly** - Send multiple reminders
✅ **Be flexible** - Have backup movies if group wants changes
✅ **Over-prepare snacks** - Better to have leftovers than run out
✅ **Take breaks seriously** - Marathon movies for 14+ hours requires bathroom/rest breaks
✅ **Have fun with it** - This is about enjoying Nic Cage movies together!
✅ **Keep energy up** - Coffee in morning, energy drinks/snacks in afternoon
✅ **End on a high note** - Schedule crowd favorites last so people leave happy
